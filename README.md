# homebrew-proto-mcp

Homebrew tap for [proto-mcp](https://github.com/just-an-oldsalt/proto-mcp).

```sh
brew tap just-an-oldsalt/proto-mcp
brew install --cask proto-mcp
```

This tap is maintained automatically by [bootstrap-tap.sh](https://github.com/just-an-oldsalt/proto-mcp/blob/main/scripts/bootstrap-tap.sh)
in the main proto-mcp repo. Don't edit `Casks/proto-mcp.rb` here
directly — the canonical source is `Formula/proto-mcp.rb` in the
main repo, copied here on every release.

See the main repo for documentation, install steps, and security
posture.
